# Phase 7 Structured Session Worksheet

Do not enter names, account IDs, room IDs, session IDs, IP addresses, quotes,
free text, audio, or drawing content. Use one sheet per three-person round.
Rotate roles across later rounds. Blank fields mean “not collected,” never an
inferred answer.

## Setup checklist

- [ ] Three friends
- [ ] Three isolated devices/browser profiles
- [ ] Three separate headsets
- [ ] Landscape layout checked with no horizontal overflow
- [ ] Roles assigned once each
- [ ] Facilitator non-coaching rule read aloud
- [ ] Recipe visible only to Recipe Keeper

role used at least one permitted role action or signal; it does not estimate

## Completion checklist

- [ ] Three structured exports collected
- [ ] Export keys match the documented allowlist
- [ ] No prohibited identity, room, free-text, audio, or drawing data present
- [ ] Local records cleared on all three devices after confirmed collection
- [ ] Roles scheduled to rotate for the next round
- [ ] No human response was filled from automation or facilitator inference

Human gate result: `PENDING` / `EVIDENCE_COMPLETE`

Use `EVIDENCE_COMPLETE` only after several rounds, full role rotation, and any
accepted balance/UI change has been retested.
